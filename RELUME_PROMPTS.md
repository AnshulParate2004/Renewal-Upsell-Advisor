# Relume Prompts - Revenue Navigator

## Main Design System
```
Neo-brutalist SaaS: purple (#8b5cf6), bold uppercase (font-900), thick borders (2-4px), block shadows (3-6px offset), white cards, purple accents. Style: Brutalist, Block Shadow, Enterprise SaaS.
```

## Landing Page
```
Neo-brutalist landing: hero (text-6xl, uppercase), sticky nav (thick border), animated banner (purple bg), CTAs (block shadows 6px), features grid, white bg with purple gradients. Style: Brutalist, Purple Theme.
```

## Dashboard
```
Data dashboard: header (text-5xl, border-bottom 4px), 6-column metric cards (responsive), white cards (borders, shadows), icons top-right, large values (text-3xl), chart below. Colors: purple primary, red alerts, green success. Style: Analytics, Brutalist Cards.
```

## Accounts Table
```
Account table: header (text-5xl, badge), toolbar (search, filters), white card table, sticky purple header (#818cf8, white text), hover effects, status badges (red/green), progress bars. Style: Data Table, Brutalist.
```

## Detail Page
```
Account detail: header (text-5xl, badge), 2-column card grid (responsive), white cards (borders, shadows), large metrics (text-4xl), sections (Metrics, Contact, Health, Risk), color coding (red/green/purple). Style: Card Layout, Brutalist.
```

## Sidebar Nav
```
Sidebar: dark purple bg (#4c1d95), white text (uppercase, font-black), active purple (#8b5cf6), hover transitions, right border (4px). Style: Dark Purple, Brutalist Menu.
```

## Buttons
```
Buttons: primary (purple bg, white text, border 2px, shadow 6px), secondary (white bg), uppercase (font-900), sizes (px-4/6/10, py-2/3/5), hover scale. Style: Block Shadow, Brutalist.
```

## Badges
```
Badges: sticker (transparent, border 2px), solid (colored bg, white text), uppercase (font-black, text-xs), status colors (red/green/yellow), pulsing dot. Style: Outline Badges, Brutalist.
```

## Charts
```
Charts: white card (border, shadow, p-6), uppercase header (font-black), purple primary (#8b5cf6), thick stroke (2px), tooltips (white bg, border), legend (uppercase). Style: Data Viz, Brutalist.
```

## Empty/Error States
```
Empty/Error: white card (centered, p-12), large icon (w-20, colored bg), title (text-3xl, uppercase), message (gray), button (primary style). Style: Centered, Brutalist.
```

## Loading
```
Loading: spinner (purple #8b5cf6, w-8/12), text (uppercase, font-black), skeleton (white bg, border, shimmer), centered layout. Style: Spinners, Brutalist.
```

## Colors
```
Primary: #8b5cf6 | Accent: #818cf8 | Background: #f8f7ff | Success: #10b981 | Destructive: #dc2626 | Warning: #f59e0b
```

## Typography
```
Font: Inter/Outfit | Weights: 400-900 | Headings: font-black, UPPERCASE, tracking-tight | Labels: font-black, UPPERCASE, tracking-wider, text-xs | Sizes: Hero (6xl-7xl), Title (5xl), Section (3xl)
```
